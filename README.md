# 💧 WaterWise – Daily Water Intake Tracker

## 📌 Project Overview
WaterWise is a mobile application designed to help users maintain healthy hydration habits by tracking their daily water intake. The app provides an intuitive interface for setting hydration goals, logging water consumption, managing reminders, viewing daily history, and analyzing weekly progress.

This project was designed in **Figma** as a high-fidelity interactive prototype following modern UI/UX principles.

---

## 🎯 Objective
The objective of this project is to create a clean, user-friendly, and visually appealing mobile application that encourages users to drink enough water every day through simple tracking and timely reminders.

## 🎨 Design System

- **Frame Size:** iPhone 14 (390 × 844 px)
- **Typography:** SF Pro Display
- **Primary Color:** Sky Blue
- **Secondary Color:** Aqua
- **Background:** White & Light Gray
- **Components Created:**
  - Primary Button
  - Quick Add Button
  - Drink Entry Card
  - Navigation Bar
- Auto Layout used for reusable layouts and history list.
- Interactive prototype created using Smart Animate transitions.
